## Stack

- **Frontend:** Next.js 14 + Tailwind CSS → Deploy: Vercel
- **Backend:** FastAPI (Python) → Deploy: Railway
- **AI Core:** PyTorch + HuggingFace Transformers
- **LLM:** Groq API (Llama 3.1)
- **Database:** Supabase (PostgreSQL)

## Quick Start

```bash
# Backend
conda activate vaic2026
cd backend
cp .env.example .env  # Điền API keys
uvicorn main:app --reload

# Frontend
cd frontend
npm install
npm run dev
```
